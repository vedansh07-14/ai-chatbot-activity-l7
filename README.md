# NOVA X — Multi-Layered Conversational Intelligence

NOVA X is a real-time, emotionally aware AI assistant designed to feel alive. It utilizes a multi-layered cognitive architecture to prioritize natural conversation, rhythm, and multimodal understanding.

## 🧠 Architecture: The Three Layers

1.  **PLANNER**: Decides *what* to say. Breaks down complex intent into conversational chunks to avoid robotic monologues.
2.  **SPEAKER**: Decides *how* to say it. Uses natural human rhythm, fillers ("hmm", "okay"), and punctuation-based pauses.
3.  **OBSERVER**: Monitors user emotion and engagement, adapting tone and speed instantly (Energized, Focus, Calm, or Friendly modes).

## ✨ Key Features

- **Multimodal Vision**: Copy an image from Finder and paste it directly into the chat. NOVA X will automatically switch to **Llama 3.2 Vision** to analyze what it sees.
- **Human Timing Engine**: Simulates cognitive processing with variable delays, making the interaction feel reactive rather than scripted.
- **Voice-First Design**: Optimized for Text-to-Speech (TTS) and Speech-to-Text (STT) for a hands-free experience.
- **Premium Dark Aesthetic**: A high-end, responsive interface with fluid animations and glassmorphism.

## 🚀 Getting Started

1.  **Configuration**: Add your Hugging Face API key to the `.env` file (or `config.js` for local labs).
2.  **Deployment**: Optimized for Vercel Serverless Functions. Run `vercel dev` locally to test the vision model switching.
3.  **Interaction**:
    - `Cmd+V` / `Ctrl+V` to paste images.
    - Click the 🎙️ icon to speak.
    - Press `Enter` to send.
