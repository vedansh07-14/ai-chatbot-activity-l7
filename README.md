# NOVA X — Multi-Layered Conversational Intelligence

NOVA X is an advanced AI chatbot application powered by Hugging Face models, featuring a sleek modern UI and multi-modal capabilities.

## ✨ Features

- **AI Chat**: intelligent text generation with human-like pacing and tone.
- **AI Image Gen**: Generate stunning visuals using Stable Diffusion XL 1.0.
- **Local Image Upload**: Attach images from your computer to use in the conversation.
- **Voice Intelligence**: Speak directly to NOVA X with integrated speech-to-text.
- **Full-Width Interface**: A modern, expansive prompt bar for a superior user experience.

## 🛠 Tech Stack

- **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript (ES6+).
- **Backend**: Vercel Serverless Functions.
- **APIs**: Hugging Face Inference API (Trinity-Large & SDXL 1.0).

## 🚀 Getting Started

1. Set your `HF_API_KEY` in your environment variables.
2. Run locally or deploy to Vercel.
